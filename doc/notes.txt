Git commands:

	git init
	git clone /path/to/rep directory/name
	git status
	git add -A
	git commit -m "message"
	git remote add origin path/to/rep (remember to add SSH insted)
	git remote -v
	git push origin master 
	git pull origin master
	ssh -T git@github.com

Bower commands:

	bower init
	bower search name
	bower info name
	bower install name#1.2.1 --save 
	bower install (read config as install)


Linux commands:

	rm -rf directory
