Install

	1. Go to "back-end/NodeExpressServer/";
	2. run "npm install";
	3. Go to "front-end";
	4. run "bower install";

Git commands:

	git init
	git clone /path/to/rep directory/name
	git status
	git add -A
	git commit -m "message"
	git remote add origin path/to/rep (remember to add SSH insted)
	git remote -v
	git push origin master 
	git pull origin master
	git diff --name-only
	ssh -T git@github.com

Bower commands:

	bower init
	bower search name
	bower info name
	bower install name#1.2.1 --save 
	bower install (read config as install)

Npm commands:

	npm init
	npm install name --save
	npm install (read config as install)

Linux commands:

	rm -rf directory
